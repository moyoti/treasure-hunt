## 项目概述

**寻宝记 (Treasure Hunt)** - 一个基于地理位置的虚拟寻宝游戏应用

| 端 | 技术栈 | 状态 |
|---|--------|------|
| Web | Next.js 14 + React + Leaflet | 🔧 开发中 |
| iOS | React Native (Expo) | 📋 待开发 |
| Android | React Native (Expo) | 📋 待开发 |
| Backend | NestJS + TypeORM + PostgreSQL | ✅ 基础完成 |

## 协作流程（强制遵循）

### 阶段1: 需求分析 (项目经理)
1. 分析用户需求
2. 拆解任务
3. 分配给对应工程师

### 阶段2: UI/UX设计 (设计师)
1. 设计界面原型
2. 确定交互流程
3. 输出设计规范给前端

### 阶段3: 开发阶段 (前后端工程师)
- 前后端工程师并行开发
- 每次代码修改后必须自测
- 发现问题立即修复，不累积

### 阶段4: QA验证（强制环节）
```
开发完成
    ↓
QA全面测试
    ↓
通过?
  ├─ 是 → 项目经理验收 → 完成
  └─ 否 → 详细Bug报告 → 开发修复 → 重新测试
```
